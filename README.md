# GuessingGame-JavaSocket


Number guessing game in Java using a client/server connection.
Server runs the game by creating a random number and asking the client to guess the correct number.
The client gets a limited number of guesses before the game is over. When an input is entered, server informs client with "increase/decrease" messages if it is not the correct number.
Once the game is over, the server tells if the client win or lose the game
